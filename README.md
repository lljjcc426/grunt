# Grunteon

Grunteon is the third generation of Grunt obfuscator. A JVM bytecode obfuscator written in kotlin.

This project is under development starting from November 2025.

Feel free to join our Discord server for suggestions: https://discord.gg/ysB2fMfeYW

## Roadmap

* [ ] Resource management
* [ ] Programmable config system
* [ ] Programmable process pipeline
* [ ] Class hierarchy and phantom class
* [ ] Renamers and optimizers
* [ ] String and number encryption
* [ ] Redirect and anti-static analysis
* [ ] Control-flow obfuscation
* [ ] Miscellaneous obfuscation
